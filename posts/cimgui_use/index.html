Using CIMGUI

July 25, 2021

Compile (static)

  • make static

Output will be present in current directory as libcimgui.a

Copy entire folder to wherever you wanna statically link then use the following commands in the compile command

-L ./cimgui -l:libcimgui.a

Compile (Shared Library)

  • make all

Output will be present in current directory as cimgui.so

Copy entire folder to wherever you want and then link using follwing:

-L ./cimgui -lcimgui

Using in C

Define the following on top

#define CIMGUI_DEFINE_ENUMS_AND_STRUCTS
#include "cimgui.h"

Make sure you lead to the correct path to cimgui.h
